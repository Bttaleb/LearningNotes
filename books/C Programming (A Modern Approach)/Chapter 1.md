Effective uses of C
- Learn its pitfalls
- Use tools (software) to make more reliable
	- LINT
- Use existing libraries
	- for tasks like: UI Dev., graphics, communications, DB management, networking, etc.
- Adopt sensible set of coding conventions
	- "style rule" --> more uniform, easier to read/modify code
- Avoid "tricky" complex code

# Chapter 2
C programs rely on three key language features
1. **Directives** (editing commands that modify program prior to compilation -> *#include <stdio.h>*
2. **Functions** (commands to be executed)
3. **Statements** (commands to be performed when the program runs)
