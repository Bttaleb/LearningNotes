# Market Study Vault

A system for studying price action patterns, key levels, and candle profiles.

## Structure

```
studies/
  key-levels/     ← CISD analysis after HTF key level taps
  candle-profiles/ ← PO3/AMD, OLH/OLC, weekly candle breakdowns
  setups/          ← General setup observations
screenshots/       ← Paste chart images here, link from notes
templates/         ← Obsidian templates for each study type
```

## Workflow

1. Open the relevant template (Cmd+T in Obsidian with Templater, or copy from templates/)
2. Paste your chart screenshot into `screenshots/` and embed it with `![[filename.png]]`
3. Fill in the properties and observations
4. Tag everything — this is how you'll query patterns later

## Tags

### Timeframe
`#1m` `#5m` `#15m` `#1h` `#4h` `#daily` `#weekly` `#monthly`

### Setup Type
`#cisd` `#po3` `#olh` `#olc` `#sweep` `#displacement` `#fvg` `#ob`

### Outcome
`#bullish` `#bearish` `#neutral` `#failed`

### Quality
`#clean` `#messy` `#textbook`
